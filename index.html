Troubleshooting

Q: Chat messages not showing?

→ Run setup.php to ensure database table exists

Q: Can't see messages from other person?

→ Both users must be logged in
→ Admin must select user from left sidebar
→ User must select admin from left sidebar

Q: Mobile view shows desktop layout?

→ Hard refresh browser (Ctrl+Shift+R)
→ Clear cache if issues persist

Q: Black overlay still blocking clicks?

→ Open browser DevTools (F12)
→ Refresh page (F5)
→ Check Console for JS errors
